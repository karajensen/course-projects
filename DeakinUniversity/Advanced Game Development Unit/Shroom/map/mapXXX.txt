SHROOM_TYPES       : Number of entries in SHROOM_VALUE & SHROOM_COUNT lists
SHROOM_VALUE       : list of shroom values for this game (separated by whitespace)
SHROOM_COUNT       : list of reltive count of each shroom type
SHROOM_NUM         : number of shrooms to load into game
SHROOM_MAGIC_VALUE : health value of the magic shroom
HUNTER_HEALTH      : initial health of hunter
HUNTER_COST_STEP   : cost of one step
HUNTER_COST_TURN   : cost of turning
HUNTER_COST_LOOK   : cost of looking
WORLD_ROWS         : size of world rows
WORLD_COLS         : size of world cols
